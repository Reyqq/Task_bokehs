.. _introduction:

Введение
=========

Этот проект использует библиотеку Bokeh для визуализации данных. В этом проекте реализованы несколько функций для создания различных графиков и взаимодействия с ними.

Проект содержит следующие компоненты:

- Визуализация данных на графиках.
- Отрисовка свечей и добавление специальных маркеров на графики.
- Настройка пользовательских стилей и элементов графика.

Следующие разделы содержат подробную документацию по каждому компоненту, включая примеры использования и описание функций.
