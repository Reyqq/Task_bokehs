2. Использование библиотеки

Наша библиотека предоставляет набор функций для создания и настройки графиков с использованием Bokeh. Вот общее руководство по использованию основных компонентов:

2.1. Подготовка данных

Перед созданием графика, вам нужно подготовить данные. Используйте функцию create_data_sources для преобразования вашего DataFrame в формат, подходящий для построения свечных графиков:

import pandas as pd
from your_library import create_data_sources

# Подготовка данных
df = pd.DataFrame({
    'date': ['2023-01-01', '2023-01-02', '2023-01-03'],
    'open': [10, 11, 9],
    'high': [12, 13, 11],
    'low': [9, 10, 8],
    'close': [11, 12, 10]
})
df, inc, dec, source = create_data_sources(df)

2.2. Создание графика

После подготовки данных, вы можете создать график, используя функцию create_candlestick_chart:

from Bokeh_task import create_candlestick_chart

# Создание графика
p = create_candlestick_chart(df, inc, dec, source)

2.3. Настройка стиля

Для настройки стиля графика используйте функцию style_plot:

from Bokeh_task import style_plot

# Настройка стиля
style_plot(p)

2.4. Добавление индикаторов

Вы можете добавить различные индикаторы на ваш график. Например, для добавления скользящей средней используйте:

from Bokeh_task import add_MA

# Добавление скользящей средней
add_MA(p, source, 'close', [20, 50], ['blue', 'red'])

2.5. Сохранение и отображение

Наконец, вы можете сохранить ваш график или отобразить его:

from bokeh.io import output_file, show

# Сохранение в файл
output_file("candlestick.html")

# Отображение графика
show(p)

Это базовое использование библиотеки. Для более сложных сценариев обратитесь к разделу с примерами.
