3.1. Обзор функций

В этом разделе представлен краткий обзор основных функций, используемых в проекте.

  • customize_plot_styles(p: Figure): Настраивает стили графиков, такие как линии, цвета и легенды.

  • create_data_sources(df: pd.DataFrame): Создает источники данных для графиков на основе предоставленного DataFrame.

  • process_positions(df: pd.DataFrame, positions_df: Optional[pd.DataFrame] = None, ...): Обрабатывает и визуализирует позиции на графике, включая покупку и продажу, а также добавляет маркеры.

  • create_markers_data(df: pd.DataFrame,indices_buy: List[int], ...): Создает данные для маркеров покупок и продаж на графике.

Подробное описание каждой функции вы найдете в разделе «Модуль bokehs».
